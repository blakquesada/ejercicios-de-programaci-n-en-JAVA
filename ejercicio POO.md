# Proyecto Java: Ejercicios de Programación Orientada a Objetos

Este repositorio contiene ejercicios prácticos en Java enfocados en los pilares de la programación orientada a objetos:

- 🔐 Encapsulamiento
- 🔁 Polimorfismo
- 🔗 Asociación
- 📩 Envío de mensajes entre objetos

## Objetivo

Explorar y aplicar conceptos como clases, objetos, métodos, UML y buenas prácticas de diseño en Java.  
Este proyecto forma parte de mi proceso de aprendizaje y documentación personal.
